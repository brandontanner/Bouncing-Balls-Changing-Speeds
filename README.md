# Bouncing Balls Changing Speeds

### Description

Three balls bouncing within a webpage and changing speed each time they hit the walls.

### Installation

Download the **moveBall.html** file and load it in a browser.

### Roadmap

No improvements or fixes scheduled at this time.

### Contributions

- **MIT Professional Certificate in Coding Staff:** 
    - Basic framework for a single ball moving in across the x and y axis of the webpage.
- **Brandon Tanner:** 
    - Introduction of 3 separately colored balls.
    - Edge detection and reversal of direction when a ball hits a wall.
    - Speeding up or slowing down each ball separately each time they strike a wall.
